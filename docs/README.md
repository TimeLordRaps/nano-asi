## Getting Started

### Installation
```bash
pip install nano-asi
```

### Basic Usage
```python
from nano_asi import ASI

asi = ASI()
result = await asi.run("Your task here")
```

### Advanced Configuration
```python
from nano_asi import Config

config = Config(
    consciousness_tracking=True,
    parallel_universes=5
)
```
